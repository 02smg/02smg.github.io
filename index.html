FLAGSHIP PROJECT

NearbRun (널브런)

AI 기반 개인 맞춤형 러닝 경로 추천 앱

널브런은 AI가 247만 개의 공공데이터와 22만 km의 전국 도로망 정보를 활용하여 사용자에게 가장 잘 맞는 최적의 경로를 추천해주는 앱입니다. 문제가 있는 코스를 신고하면 실시간으로 모든 경로가 수정되는 독자적인 경로 추천 알고리즘을 보유하고 있습니다.

또한 부렁이라는 캐릭터 IP를 활용하여 사용자의 서비스 리텐션을 강화할 계획입니다. 널브런은 2026년 7월 중 런칭 예정입니다.

✓
AI 최적 경로 추천: 사용자 성향과 주변 환경을 고려해 가장 잘 맞는 러닝 코스를 추천
✓
공공데이터 기반 분석: 247만 개의 공공데이터와 22만 km 전국 도로망 정보 활용
✓
실시간 경로 수정: 문제가 있는 코스 신고 시 전체 추천 경로를 즉시 업데이트
✓
캐릭터 IP 리텐션: 부렁이 캐릭터를 활용해 앱에 다시 돌아오고 싶은 경험 설계
